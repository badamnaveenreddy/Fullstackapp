\## Overview

This project is a backend application built using Spring Boot and MongoDB.

It provides REST APIs to manage projects and demonstrates full-stack backend development skills.



\## Tech Stack

\- Java

\- Spring Boot

\- MongoDB

\- Spring Data MongoDB

\- REST APIs

\- Git \& GitHub



\## Features

\- Add new project

\- Get all projects

\- RESTful API design

\- MongoDB integration



\## API Endpoints



\### Add Project

POST /api/projects



Request Body:

{

&nbsp; "name": "Student Attendance System",

&nbsp; "description": "Spring Boot + MongoDB Project",

&nbsp; "imageUrl": "https://example.com/image.png"

}



\### Get All Projects

GET /api/projects

2\. Open in Eclipse
